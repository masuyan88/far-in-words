# UI_GUIDE.md — UI 设计规则

## 整体风格

**旧纸感 + 文学地图 + 复古明信片 + 旅行手账氛围**

注意：这只是 UI 风格，项目的核心仍然是"文学地图"。

## 色彩规范

| 用途 | 变量名 | 色值 | 说明 |
|------|--------|------|------|
| 页面背景 | --background | #f3ece0 | 米白旧纸 |
| 卡片/弹窗背景 | --paper | #efe6d4 | 浅纸色 |
| 暗部纸色 | --paper-dark | #e4d8c0 | 渐变用 |
| 主文字 | --ink | #2e2318 | 深棕，主要文字 |
| 中等文字 | --ink-medium | #5a4a3c | 辅助文字 |
| 弱文字 | --ink-light | #6b5c4e | 说明、标签 |
| 强调红色 | --accent | #7a1f00 | 选中、邮戳 |
| 柔和棕红 | --accent-soft | #8b3a1a | hover 过渡 |
| 橄榄绿 | --olive | #4a5e28 | 地点标记默认色 |
| 沙色 | --sand | #c8a882 | 装饰、分割线 |
| 大陆填充 | --continent | #d4c4a0 | SVG 地图大陆 |
| 大陆描边 | --continent-stroke | #b8a880 | SVG 地图边境 |

### 禁止使用
- 亮蓝、亮绿、荧光色
- 纯白 #ffffff 作为大面积背景
- 强渐变、现代科技风元素
- 纯黑 #000000

## 字体

```css
font-family: "Noto Serif SC", "STSong", "SimSun", "Songti SC", "Georgia", serif;
```

- 优先使用衬线字体
- 中文标题：bold / 600-700 weight
- 正文：regular / 400 weight
- 装饰文字（邮戳、标签）：小号 + letter-spacing

## 文字可读性

- 主文字对比度至少 10:1（`#2e2318` on `#f3ece0`）
- 辅助文字不能淡于 `--ink-light`（#6b5c4e）
- 邮戳和装饰文字可以淡但不能不可读
- 手机端字号不小于 14px（正文）

## 卡片风格

- 使用 `.paper-card` 类（旧纸背景 + 轻微阴影 + 内发光）
- hover 使用 `.card-lift` 类（轻微上浮 + 阴影加深）
- 边框颜色 `rgba(46, 35, 24, 0.12)`
- 圆角 2-4px（模拟纸张切角）

## 按钮风格

- 主按钮：`.btn-ink`（深色背景 + 浅色文字）
- 次按钮：`.btn-paper`（透明背景 + 细边框）
- 不做圆角大按钮、渐变按钮、投影按钮

## 弹窗

- 背景半透明遮罩：`rgba(46, 35, 24, 0.5)` + `backdrop-filter: blur(3px)`
- 弹窗本体：`.paper-card` + `.paper-texture`
- 点击遮罩关闭
- 手机端最大高度 85vh 可滚动

## 响应式

| 断点 | 卡片列数 | 地图 | 弹窗 |
|------|---------|------|------|
| 手机 (<768px) | 1 列 | 全宽，点位文字缩小 | 全屏宽，85vh 高 |
| 平板 (768-1024px) | 2 列 | 全宽 | 居中，max-w-lg |
| 电脑 (>1024px) | 3 列 | 居中，max-w-[1200px] | 居中，max-w-lg |

## 装饰元素使用规则

- 邮戳：`.stamp`（仅用于卡片和弹窗装饰）
- 折线：`.fold-line`（仅用于弹窗、大卡片）
- 纸张纹理：`.paper-texture`（背景用）
- 脉冲动画：`.pulse-ring`（仅用于地图选中点位）
- 和纸胶带：`.washi-tape`（Hero 装饰）
- 明信片折角：`.postcard-corner`（卡片装饰）
