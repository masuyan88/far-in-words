# MAP_GUIDE.md — 地图实现规则

## 核心原则

地图是文学地图的核心入口。它必须保留，但**不做真实地图**。

## 禁止事项

- 不接任何地图 SDK（Mapbox、Google Maps、高德、百度、Leaflet 等）
- 不做真实可缩放地图
- 不做真实经纬度定位
- 不做导航、路线规划
- 不做地理精确的地图

## 实现方式

使用 **SVG** 在组件 `LiteraryWorldMap.tsx` 中绘制简化世界地图。

### 地图容器
- 尺寸比例：16:9（padding-bottom: 56.25%）
- 最大宽度：1200px
- 背景：旧纸色 + 纸张纹理

### SVG 内容
- 大陆块：用 `<path>` 绘制简化形状
- 经纬线：`<line>` + 弧线 `<path>`，opacity 0.06-0.08
- 边框：旧地图风格的四边线框
- 指南针：右下角装饰

### 点位样式

**未选中状态**：
- 圆点：橄榄绿 (#4a5e28) 填充 + 描边
- 标签文字：深棕，有背景纸色阴影保证可读性
- hover：圆点变为暗红 (#8b3a1a)

**选中状态**：
- 圆点：暗红 (#7a1f00) 填充 + 描边
- 发光：`box-shadow: 0 0 8px rgba(122, 31, 0, 0.4)`
- 脉冲圈：`.pulse-ring` 动画
- 标签文字：暗红色
- 整体 scale(1.1)

### 中美洲标签错开方案

洪都拉斯和巴拿马在 mapPosition 中已通过 x 偏移错开：
- 洪都拉斯 x:26（在墨西哥城 22 的右侧）
- 巴拿马 x:30（在洪都拉斯右侧）

如果仍重叠，可以通过 CSS transform 微调标签位置。

## 装饰元素

- 经纬线：淡至 opacity 0.06
- 边框：类似古地图的外框
- 指南针：右下角，简约 N/S 标注
- 折叠线：模拟纸张折痕

## 交互

- 点击点位 → 更新 selectedPlaceId
- hover → 圆点颜色过渡 + 轻微放大
- 选中点位 → 脉冲动画（持续）
- 不接受拖拽、缩放

## 后续可优化

- 大陆形状可以逐渐调优更像真实地图
- 可以增加旅行路线虚线连接
- 可以增加更多装饰性图例
